# Taskify - Fullstack (Backend + React Frontend)

## Setup

### Backend
- cd backend
-`.env` and set `MONGO_URI` (defaults to mongodb://localhost:27017/taskifydb)
- npm install
- npm run dev (requires nodemon) or npm start

### Frontend
- cd frontend
- npm install
- npm start 

The frontend expects backend at http://localhost:5000/api/tasks. Backend serves API at /api/tasks.
